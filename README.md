# Sober Helper — MiniApp + Web/PWA (Starter)

Этот репозиторий подготовлен как пустая точка старта под новый проект Sober Helper: единая кодовая база для Telegram Mini App и полноценного веб/PWA с поддержкой RU/EN. Aiogram-бот не нужен (можно добавить уведомления отдельным сервисом позже).

## Что здесь есть
- **CODEX_PROMPT.md** — готовый промпт для Codex/LLM, который сгенерирует структуру фронтенда и бэкенда, базовые экраны, модели и TODO-листы.
- Этот README — краткое описание целей и ожидаемого результата.

## Как использовать промпт
1. Открой `CODEX_PROMPT.md` и скопируй весь текст.
2. Вставь его в Codex/генератор кода с инструкцией «создай проект в этом репозитории».
3. Сгенерированный код заменит пустой каркас и создаст единый проект MiniApp + Web/PWA (React + Tailwind + FastAPI/Node на выбор, PostgreSQL, OpenRouter).

## Минимальный функционал, который должен собрать генератор
- Приветствие/Login (Telegram login для MiniApp + email/пароль для Web).
- Онбординг с данными о зависимости, мотивацией и целью.
- Главный экран со стриком, целью, быстрыми действиями.
- AI-чат на OpenRouter с history+summary и кнопкой SOS.
- Daily Check-in с вопросами о настроении/тяге/срывах.
- Прогресс: графики, streak, триггеры, аналитика ИИ.
- Журнал заметок.
- Настройки (язык, тема, цель, уведомления, лимиты).
- Монетизация (бесплатно/подписка), биллинг.
- Простая админка (пользователи, чек-ины, summary, метрики, ручные уведомления).

## После генерации
- Проверь README, который создаст Codex, и следуй его шагам запуска.
- Дополнительно закоммить TODO-лист из промпта для дальнейшей разработки.

Удачной сборки! Минималистичный дизайн в стиле Telegram, mobile-first, адаптив под десктоп.
